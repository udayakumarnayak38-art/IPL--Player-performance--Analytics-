# IPL-Player-Analytics-Dashboard
Interactive KPI dashboard analyzing IPL player performance across seasons | Built with Power BI, DAX
📌 Overview

This project is a data analytics platform built to analyze IPL player performance from 2008 to 2025 using ball-by-ball data. It provides interactive KPI dashboards to help in decision-making for team strategy, player auctions, and performance evaluation.

🎯 Objectives
Analyze batting, bowling, and fielding performance across seasons
Perform phase-wise analysis (Powerplay, Middle Overs, Death Overs)
Build composite impact scores for players
Enable dynamic filtering by:
Season
Venue
Opposition
Match phase
Identify undervalued players using performance vs auction price
📂 Dataset
Primary Data: IPL ball-by-ball dataset (2008–2025)
Secondary Data:
Player details
Team data
Venue statistics
Size:
~240,000 deliveries
500+ players
18 seasons
